FFmpeg 入门

封装和编码

命令格式

ffmpeg -i [输入文件名] [参数选项] -f [格式] [输出文件]

样例

ffmpeg -i [输入文件名] [参数选项] -f [格式] [输出文件]
ffmpeg -i a.flv -c:a copy -c:v copy -f mp4 a.mp4
